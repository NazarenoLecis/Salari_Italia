# Dizionario dati

| Campo | Significato |
|---|---|
| `source` | Ente o sistema statistico che pubblica il dato. |
| `dataset` | Codice della tavola o del dataset di origine. |
| `source_request` | Sottoinsieme richiesto dalla pipeline. |
| `year` | Anno di riferimento. |
| `geography_type` | Livello territoriale. |
| `geography_code` | Codice territoriale originale. |
| `geography_name` | Etichetta territoriale. |
| `geography_basis` | Residenza, luogo di lavoro, sede dell'impresa o paese dichiarante. |
| `sex` | Codice del sesso nella fonte. |
| `sex_label` | Etichetta del sesso. |
| `age_class` | Classe di età originale. |
| `age_label` | Etichetta della classe di età. |
| `education` | Titolo o livello di studio originale. |
| `education_label` | Etichetta del titolo di studio. |
| `occupation` | Professione secondo la classificazione della fonte. |
| `occupation_label` | Etichetta della professione. |
| `employment_status` | Posizione o stato occupazionale. |
| `contract_type` | Tipo o durata del rapporto di lavoro. |
| `working_time` | Tempo pieno, parziale o totale. |
| `working_time_label` | Etichetta del regime di orario. |
| `sector` | Settore economico. |
| `sector_label` | Etichetta del settore. |
| `firm_size` | Classe dimensionale dell'impresa. |
| `firm_size_label` | Etichetta della dimensione. |
| `public_private` | Settore istituzionale o natura pubblica/privata. |
| `pay_concept` | Retribuzione, reddito, imponibile o costo del lavoro. |
| `pay_period` | Ora, mese, anno o altro periodo. |
| `statistic` | Media, mediana, percentile, quota o rapporto. |
| `percentile` | Percentile numerico quando applicabile. |
| `measure_code` | Codice originale o armonizzato della misura. |
| `value` | Valore numerico. |
| `unit` | Codice dell'unità di misura. |
| `unit_label` | Etichetta dell'unità. |
| `population` | Numero di unità nella popolazione, quando disponibile. |
| `sample_size` | Numerosità campionaria, quando disponibile. |
| `quality_flag` | Flag di qualità della fonte. |
| `source_url` | URL usato per il download. |
| `download_timestamp` | Data di acquisizione. |
| `dimensions_json` | Dimensioni originali della cella statistica. |
